Professional Accessibility Javascript Evaluation Tools

PAJET는 자바스크립트를 이용하여 웹 페이지의 웹 접근성이 얼마나 잘 지켜 졌는지 알 수 있는데 도움을 줄 수 있는 도구 입니다. 아직 완벽하게 만들어진것은 아니지만 많은 도움을 줄 수 있다고 자부합니다.

개요

페이지의 구조를 통해 각종 앨리먼트들이 어떤 방식으로 사용되었는지 그 정보를 알려줍니다.

구현항목

  • 1.1.1 적절한 대체 텍스트 제공
  • 1.1.1 적절한 대체 텍스트 제공(배경사용여부)
  • 2.4,1 반복영역 건너뛰기
  • 2.4.2 제목 제공(페이지 타이틀)
  • 2.4.2 제목 제공(프레임제목)
  • 2.4.2 제목 제공(콘텐츠 블록)
  • 3.1.1 기본언어 표시
  • 3.3.2 표의 구성
  • 3.4.1 레이블제공
  • 4.2.1 웹 어플리케이션 접근성 준수(서식의 올바른 사용)
  • 4.2.1 웹 어플리케이션 접근성 준수( pseudo자바스크립트 사용 여부)
  • 4.2.1 웹 어플리케이션 접근성 준수(onclick 사용시 대체 URL체크)

구현해야 할것

  • object 앨리먼트에 대한 정보 노출

알려진 문제점

  • 프레임으로 구성된 페이지는 body가 없기 때문에 PAJET이 작동하지 않습니다.

주의 할 점

  • 이 툴은 웹 페이지에 대한 웹 접근성 자체를 평가해주는 툴은 아닙니다.
    단지 웹 접근성에 대한 충분한 이해를 한 사람이 페이지에 대한 접근성을 체크하는데 도움을 주기 위한 툴 일뿐 입니다.
    접근성 평가를 해주는 툴은 다른데서 알아보세요.
  • 이 툴을 100% 과신해서는 안됩니다. 결과 오류에 대한 책임을 저는 지지 않습니다.

사용 방법

  1. 다음의 링크를 오른쪽 클릭하시고 즐겨찾기(연결)에 추가를 하시면 됩니다.

    PAJET 2.0 실행

  2. 툴을 실행하고자 하는 페이지로 이동하여 즐겨찾기(연결)에 있는 "PAJET 2.0 실행" 링크를 클릭하여 활성화합니다.

저작자

PAJET은 신현석님과 홍윤표가 주도해서 만듭니다만 아무나 개발에 참여 할 수 있습니다. 소스의 특성상 직접가져가셔서 개발을 진행하셔도 됩니다만 재 배포 시에 저희에게 알려주시기 바랍니다. 좋은것 있으면 저희도 차용하게요.
